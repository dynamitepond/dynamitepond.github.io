About

Dynamite Pond is a static website generated using Hugo and hosted on GitHub Pages. It is designed to be minimal and easy to navigate.

Layout and style

The layout has been kept very simple, to improve compatibility with many devices. The styling is also minimal, no animations or pop ups to frustate a reader with delays and hiding menus.

Advertising

At the moment there are no advertisements on this site, and I will never include any javascript/markup from a third party advertising company. If anyone is interested in advertising, contact me and we can discuss it. I am open to including discrete advertisements and/or links in the markup for this site.

Privacy

You can expect:

  • No cookies
  • No javascript
  • No analytics or trackers
  • No one carefully watching your every move1

  1. GitHub is a third party serving the static website. In theory, they can record every web request made. ↩︎
